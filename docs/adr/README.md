# ADRs do Spider Code Analysis Platform

Este diretório contém 50 ADRs (Architecture Decision Records) para orientar a evolução do projeto de scripts isolados para plataforma robusta.

## Convenções
- **Status**: Proposed / Accepted / Superseded / Deprecated.
- **Estrutura**: Contexto, Decisão, Consequências, Plano de implementação.
- **Versionamento**: Um ADR por arquivo com numeração estável.
